# Minikoban
A little Sokoban game for Android

## Why

1. Because it's a fun game
2. To see how small I can make it (apk size)

## General format for public files
```
Wall	                #
Player	                @
Player on goal square	+
Box	                    $
Box on goal square	    *
Goal square	            .
Floor	             (Space)
Floor                   -
```
